# Smart Expense Tracker

Simple Python CLI application for tracking expenses by category.

## Features
- Add expenses
- Calculate total
- Show highest spending category

## Technologies
- Python 3

## How to Run
1. Clone the repo: `git clone https://github.com/ljubaniclazar-bit/smart-expense-tracker.git`
2. Run `main.py` with Python 3
